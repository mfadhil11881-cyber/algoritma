Algoritma Distribusi Frekuensi

Langkah-langkah algoritma:

  1. Input data - Menerima data numerik dari pengguna
  2. Menentukan jumlah kelas (k) - Menggunakan aturan Sturges: k = 1 + 3.3 * log(n)
  3. Menentukan interval kelas (c) - c = (max - min) / k
  4. Menyusun tabel distribusi frekuensi - Menghitung frekuensi data pada setiap interval kelas
  5. Menampilkan hasil - Tabel distribusi frekuensi dan visualisasi
